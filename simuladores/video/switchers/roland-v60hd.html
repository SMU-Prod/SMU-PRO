MEMORY — salvar/recuperar cenas
A V-60HD é plug-n-play: as 6 entradas já entram configuradas. Você opera direto no painel — escolhe a próxima fonte no PST, transiciona com AUTO/CUT ou o fader, e usa PinP, KEY e SPLIT para compor. As MEMORY guardam a cena inteira para chamar num toque. O RCS faz tudo isso pela rede, de longe.
