MPC = Music Production Center. O fluxo é sempre o mesmo desde 1988: PROGRAM (o kit de sons nos pads) → SEQUENCE (o loop que você grava) → SONG (as sequences encadeadas = a música). Grave com REC + PLAY, corrija o tempo no TIMING CORRECT, troque o som de um pad no PROGRAM e esculpa com os Q-LINK. Arquivos: programa .XPM, sample .WAV, sequência .SXQ, projeto .XPJ.
