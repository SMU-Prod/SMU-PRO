import { NextRequest, NextResponse } from "next/server"; import { auth } from "@clerk/nextjs/server"; import { GoogleGenAI } from "@google/genai"; import { createAdminClient } from "@/lib/supabase/server"; import { tts } from "@/lib/edge-tts"; const ai = new GoogleGenAI({ apiKey: process.env.GOOGLE_GENAI_API_KEY }); // ── Passo 1: Gemini gera o roteiro detalhado ── async function generateScript(titulo: string, conteudo: string, categoria: string) { const response = await ai.models.generateContent({ model: "gemini-2.5-flash", contents: `Você é um diretor de animações didáticas para cursos de eventos ao vivo. Analise o conteúdo abaixo e gere um ROTEIRO DE ANIMAÇÃO em JSON. REGRAS CRÍTICAS: - Entenda PROFUNDAMENTE cada conceito antes de criar as cenas - Cada cena deve ter: titulo, narracao (texto natural em PT-BR), prompt_imagen (prompt detalhado em inglês para gerar a ilustração), duracao (4-8 segundos), tipo (abertura|conceito|exemplo|alerta|pratica|resumo) - Os prompts de imagem devem ser ULTRA ESPECÍFICOS para o contexto de eventos - Inclua referências visuais reais: equipamentos, marcas, cenários de shows/festivais - Gere entre 8-12 cenas para cobrir todo o conteúdo - A narração deve ser didática, como um professor conversando Categoria: ${categoria} Título: ${titulo} Conteúdo: ${conteudo} Responda APENAS com o JSON válido no formato: { "cenas": [ { "numero": 1, "tipo": "abertura", "titulo": "...", "narracao": "...", "prompt_imagen": "... detailed English prompt for Imagen ...", "duracao": 5 } ] }`, }); return JSON.parse(response.text); } // ── Passo 2: Imagen gera as ilustrações ── async function generateSceneImages(cenas: any[]) { const results = []; for (const cena of cenas) { const response = await ai.models.generateImages({ model: "imagen-4.0-fast-generate-001", prompt: cena.prompt_imagen, config: { numberOfImages: 1, aspectRatio: "16:9", }, }); results.push({ ...cena, imageBytes: response.generatedImages[0].image.imageBytes, }); } return results; } // ── Passo 3: Veo anima as imagens (image-to-video) ── async function generateSceneVideos(cenas: any[]) { const operations = []; for (const cena of cenas) { const op = await ai.models.generateVideos({ model: "veo-2-generate-001", prompt: `Smooth cinematic animation: ${cena.prompt_imagen}. Camera slowly zooms in with gentle parallax movement.`, image: { imageBytes: cena.imageBytes, mimeType: "image/png" }, config: { aspectRatio: "16:9" }, }); operations.push({ cena, operation: op }); } // Poll all operations until complete for (const item of operations) { while (!item.operation.done) { await sleep(10000); item.operation = await ai.operations.getVideosOperation({ operation: item.operation, }); } item.videoFile = item.operation.response.generatedVideos[0].video; } return operations; } // ── Passo 4: TTS gera as narrações (já existe no projeto) ── async function generateNarrations(cenas: any[]) { return Promise.all( cenas.map(cena => tts(cena.narracao)) ); } // ── Passo 5: Upload tudo para Supabase Storage ── async function uploadToStorage(lessonId: string, videos: any[], narrations: Buffer[]) { const supabase = createAdminClient(); const urls = []; for (let i = 0; i < videos.length; i++) { // Download video from Google const videoData = await ai.files.download({ file: videos[i].videoFile }); // Upload video const vPath = `animations/${lessonId}/scene-${i + 1}.mp4`; await supabase.storage.from("media").upload(vPath, videoData); // Upload narration audio const aPath = `animations/${lessonId}/narration-${i + 1}.mp3`; await supabase.storage.from("media").upload(aPath, narrations[i]); urls.push({ video: vPath, audio: aPath }); } return urls; } // ── POST handler ── export async function POST(req: NextRequest) { const { userId } = await auth(); if (!userId) return NextResponse.json({ error: "Não autenticado" }, { status: 401 }); const { lessonId, titulo, conteudo, categoria } = await req.json(); const supabase = createAdminClient(); // Check cache const { data: cached } = await supabase .from("ai_animations").select("*") .eq("lesson_id", lessonId).limit(1); if (cached?.length) return NextResponse.json(cached[0]); // Set status to "generating" await supabase.from("ai_animations").upsert({ lesson_id: lessonId, status: "generating", created_at: new Date().toISOString(), }); try { // Pipeline completo const script = await generateScript(titulo, conteudo, categoria); const withImages = await generateSceneImages(script.cenas); const withVideos = await generateSceneVideos(withImages); const narrations = await generateNarrations(script.cenas); const urls = await uploadToStorage(lessonId, withVideos, narrations); // Save to DB const record = { lesson_id: lessonId, status: "ready", roteiro: script, urls, model: "gemini-2.5-flash+imagen4+veo2", }; await supabase.from("ai_animations").upsert(record, { onConflict: "lesson_id" }); return NextResponse.json(record); } catch (error) { await supabase.from("ai_animations").update({ status: "error" }) .eq("lesson_id", lessonId); return NextResponse.json({ error: "Falha na geração" }, { status: 500 }); } }
⚖️ Duas Estratégias de Implementação
🟢 Estratégia A: Imagen + CSS/JS (Econômica)
$0.22/animação Geração: ~40s Sem Veo
Como funciona:
Gemini gera roteiro → Imagen gera 10 ilustrações HD → Player no browser anima as imagens com efeitos CSS (Ken Burns zoom, parallax, fade, slide) + overlay de texto + narração TTS sincronizada.

Prós: Custo quase zero, geração rápida (~40s), funciona em qualquer device, sem armazenamento pesado.
Contras: Animação limitada a efeitos de câmera sobre imagens estáticas (não tem movimento real dos elementos).

Ideal para: Cursos gratuitos, conteúdo teórico, explicações conceituais.
🟡 Estratégia B: Imagen + Veo (Premium)
$21/animação Geração: ~5min Com Veo 2
Como funciona:
Gemini gera roteiro → Imagen gera ilustrações → Veo 2 transforma cada imagem em vídeo animado com movimento real → Concatena com narração TTS → Upload para Supabase.

Prós: Animação cinematográfica com movimento real, qualidade de vídeo explicativo profissional, diferencial enorme no mercado.
Contras: Custo mais alto por animação, tempo de geração maior, armazenamento de vídeo.

Ideal para: Cursos pagos premium, conteúdo prático/demonstrativo, marketing.
💡 Recomendação: Implementar as DUAS estratégias
Comece pela Estratégia A (rápida, barata, valida o conceito). Depois adicione a Estratégia B como upgrade premium. No admin, o botão "Gerar Animação" pode ter duas opções: "Rápida (imagens)" e "Premium (vídeo)".
